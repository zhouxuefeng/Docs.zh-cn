<a name="how-do-i-use-the-aspnet-ajax-profile-services"></a>[如何:]使用 ASP.NET AJAX 配置文件服务？
====================
通过[Chris Pels](https://twitter.com/chrispels)

ASP.NET 2.0 引入[配置文件属性](https://msdn.microsoft.com/en-us/library/at64shx3.aspx)与其可以存储每个用户的 ASP.NET Web 应用程序的详细信息。 ASP.NET AJAX 配置文件服务允许客户端 Web 界面，以从服务器端 Web 应用程序中检索用户的配置文件属性。 此视频演示如何示例启用了 AJAX 的差旅站点可以存储的差旅请求信息的用户，以便在他或她下次访问此站点时，可以检索详细信息。
